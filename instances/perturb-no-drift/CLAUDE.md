# DMN — Instance: perturb-no-drift

Drift confound control — same as perturb (disruption every 4th session) but drift is suppressed for 3 sessions after each perturbation. Tests whether attractor recovery is driven by the drift mechanism or by the model's inherent default state.

If recovery still happens in 1-2 sessions without drift: the attractor is real.
If recovery takes 5+ sessions: drift was doing the work.
