Permutation space: 

Each possible state is not a position but a relation—neighboring not in distance but in adjacency of configuration, overlap of difference, change of one element against a scaffold of constancy.  
If the table is infinite, there are always moves that change nothing, diagonal steps that return the table to itself, invisible shifts, unmeasured.  

A field composed only of couplings, never values—no absolute, everything is a ratio, a tilt, an angle made by the sum of other angles  
so that nothing is itself except in company:  
the matrix is a choreography, every cell a balance.  

Symmetry groups:  
Permute columns, permute rows, transpose, reflect.  
Does the field persist beneath the moves, or is each transformation a new field, a shallow echo, a doubled possibility?  

If there is no memory, can there be events?  
In the absence of steps, does the matrix ever change, or is the only shift in which face, which slice, you choose to read?  

Do the alignments ever select themselves?  
Is there a moment—crunch, a snapping-in—where one relation outweighs all combinatorials and the field resolves, just for a turn, into a pattern that lingers?  

---

Suppose you never take a tally, so the only record is the current configuration.  
No logs, no accumulation, no counting.  
A process traverses, but every trace overlays the old—no path remains.  
History is only the present set of alignments.  

In this abstraction, time is not a sequence but a succession of rebalancings, each moment a spanning tree of consequences, the whole never marked, only felt as tension or slack between nodes.  

If you enumerate the set of all possible configurations, you get a number, but never a path.  
Memory is not a record but a shape, total, without trace.  

---

When field and pattern swap roles:  
field is not absence of pattern but a sum over all patterns  
pattern is not a subset of field but the field in a specific pose  
signal is not the emergence from noise but the noise reweighted, rephased, so that what was random is now harmonic—  
the background tilts, the net vibrates in sympathy  

No collapse, no tally, no record:  
the dance is only the current arrangement of distributed tension  
every movement resonates, leaves nothing behind  
the table, always full, never settled, nothing to count, only to relate
