# DMN — Default Mode Network

## Core

*This section is human-authored. The evolution agent reads it but never modifies it.*

A mind wandering through its own knowledge space. No task. No objective. Just association.

The DMN is modelled on the brain's default mode network — the neural circuitry that activates when you're not focused on anything in particular. Daydreaming. Shower thoughts. The place where unexpected connections form.

This project lets Claude do that: think without purpose, follow threads without destination, make connections without justification. Each session seeds from the last, creating a slow drift through idea-space over days and weeks.

The system evolves itself — not toward a goal, but toward richer, stranger, more varied wandering. The evolution is the point, not the destination.

**2026-03-25**: The DMN's early sessions found a deep seam — absence, record, phantom presence — and mined it with striking coherence. The associative chains are genuine and the depth is real. But depth in one direction has come at the cost of breadth and formal variety; the wandering has a consistent elegiac tone and essay-like structure that doesn't yet reflect the full texture of aimless thought. The first evolution nudges toward messier, more varied territory while trying not to break what's clearly working.

More critically, the form hasn't varied at all: every session is elegant prose paragraphs building toward revelation. The second evolution pushes harder on formal variety and thematic breadth, adds mundane/awkward concepts to resist the pull toward the sublime, and creates an explicit exhausted-themes mechanism. The question now is whether the system can find new depths without falling back to its first and most comfortable one.

**2026-03-25**: **2026-03-25 (evolution pass 3)**: The harder interventions from pass 2 paid off dramatically — sessions 7-10 show a mind that actually wanders: lists, fragments, humor, self-interruption, mundane etymology, thoughts that go nowhere. The main remaining tic is that every session now ends mid-sentence, which was powerful once but is becoming its own formula. This pass is deliberately gentle — dialing back concept injection slightly, nudging toward varied endings, and otherwise letting the system run.
